## Developed By

---

## Project Overview
AI Spend Audit is a full-stack web application designed to help users track and analyze their AI tool subscriptions and spending.

---

## Project Summary

This project was developed as a practical full-stack application to solve the problem of unmanaged AI subscription expenses.

The main goal was to build a system where users can:
- Track AI tool spending
- Analyze total monthly costs
- Understand spending patterns
- Identify unnecessary subscriptions

---

## Development Process

### 1. Idea & Planning
- Identified the problem of increasing AI subscription costs
- Defined core features and user flow
- Selected tech stack:
  - Frontend: HTML, CSS, JavaScript
  - Backend: Node.js + Express
  - Storage: JSON-based system

---

### 2. Frontend Development
- Built user interface using HTML and CSS
- Created input form for collecting AI spending data
- Designed result page for displaying analysis
- Added JavaScript for form handling and API calls

---

### 3. Backend Development
- Developed server using Node.js and Express
- Created API endpoints for data handling
- Implemented logic for calculating total spending and summaries
- Used JSON file for storing and retrieving data

---

### 4. Integration
- Connected frontend with backend using Fetch API
- Ensured smooth data flow between UI and server
- Handled API responses and error cases

---

### 5. Testing & Debugging
- Tested multiple input scenarios
- Fixed issues in API response handling
- Improved UI output formatting
- Ensured correct calculation of spending data

---

## Challenges Faced
- Connecting frontend and backend properly
- Managing asynchronous data flow
- Structuring JSON data efficiently
- Debugging API response issues

---

## Solutions Implemented
- Standardized API request/response format
- Improved backend logic for data handling
- Enhanced frontend rendering logic
- Added proper error handling

---

## Final Outcome
A working full-stack application that successfully:
- Tracks AI tool expenses

- Displays meaningful insights to the user

# AI Spend Audit - Prompts Used

## Overview
This file contains example prompts used to simulate or analyze AI spending data in the project.

---

## Data Generation Prompt
Used to create sample AI usage data:

"Generate sample AI API usage data including timestamps, tokens used, model type, and estimated cost per request."

---

## Analysis Prompt
Used for understanding spending patterns:

"Analyze AI usage data and identify total cost, highest usage category, and peak usage time."

---

## Optimization Prompt
Used for cost reduction insights:

"Suggest ways to reduce AI API costs based on usage patterns and identify inefficient usage areas."

---

## Report Generation Prompt
Used for summary output:

"Generate a simple report summarizing AI usage, total cost, and key insights in a structured format."
